# Getting Started

This guide walks you through installing Biz UI and configuring your project.

## Installation

Install the package and its peer dependencies. Biz UI requires React, Radix UI primitives (used internally), and styling utilities.

### npm

```bash
npm install biz-ui react react-dom
```

### yarn

```bash
yarn add biz-ui react react-dom
```

### pnpm

```bash
pnpm add biz-ui react react-dom
```

### bun

```bash
bun add biz-ui react react-dom
```

## Peer dependencies

For full functionality, ensure these peer dependencies are installed. Most are included when you install `biz-ui`; add any that your project does not already have:

- `react` and `react-dom` (^18.0.0 or ^19.0.0)
- `tailwindcss` and `tailwindcss-animate`
- `class-variance-authority`, `clsx`, `tailwind-merge`
- `lucide-react` (for icons)
- `date-fns` (for date components)
- Radix UI packages (accordion, dialog, dropdown, etc.) — see the [biz-ui package.json](https://github.com/biz-ui/biz-ui/blob/main/packages/ui/package.json) for the full list

## Tailwind CSS

Biz UI is styled with Tailwind. Your app must use Tailwind and include the library’s styles.

1. **Import Biz UI styles** in your app entry (e.g. `main.tsx` or `App.tsx`):

```tsx
import "biz-ui/styles";
```

2. **Configure Tailwind** so it can scan the library’s classes. In `tailwind.config.js`:

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,astro,mdx}",
    "./node_modules/biz-ui/dist/**/*.js",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

Adjust the second path if your `node_modules` or build output is elsewhere.

## Theme provider

Wrap your app (or the part that uses Biz UI) with `OceanThemeProvider` and pass the default theme. Use a wrapper with class `biz-ui` so component styles apply correctly.

```tsx
import { OceanThemeProvider, defaultTheme } from "biz-ui";
import "biz-ui/styles";

function App() {
  return (
    <OceanThemeProvider theme={defaultTheme}>
      <div className="biz-ui">
        {/* Your app and Biz UI components */}
      </div>
    </OceanThemeProvider>
  );
}
```

You can pass a custom theme to `OceanThemeProvider` to override tokens (colors, spacing, etc.).

## First component

You’re ready to use components:

```tsx
import { Button } from "biz-ui";

export function MyComponent() {
  return <Button>Click me</Button>;
}
```

For more examples and props, see the [component documentation](/docs/components/button) and [Storybook](https://github.com/biz-ui/biz-ui#storybook).
